GESTION FORESTAL LUBAYD SA - VERSION 5.0

Esta versión conserva toda la funcionalidad anterior y mejora especialmente la experiencia en celular:
- Diseño más cálido, claro y amigable.
- Tarjetas con colores suaves y lectura rápida.
- Botones más grandes y cómodos para tocar.
- Formularios más simples y mejor adaptados a iPhone.
- Navegación inferior tipo aplicación.

Para publicar:
1. Sube todos los archivos al repositorio LUBAYD-SA.
2. Reemplaza los archivos anteriores.
3. Espera la actualización de GitHub Pages.
4. En iPhone, cierra y vuelve a abrir la app. Si continúa la versión anterior, elimina el acceso de inicio y agrégalo nuevamente desde Safari.

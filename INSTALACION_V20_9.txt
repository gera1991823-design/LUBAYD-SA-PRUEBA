LUBAYD SA V20.9.0 - INSTALACIÓN

1. Subir todos los archivos del ZIP completo a la raíz de GitHub.
2. En Cloud Shell ejecutar:
   cd ~/LUBAYD-SA-PRUEBA
   git pull origin main
   npm --prefix functions install
   firebase deploy --only "functions:syncFuelRecord" --project lubayd-sa
3. No borrar la PWA si existen registros pendientes.
4. Actualizar la aplicación desde Más > Configuración.
5. Confirmar que la aplicación indique V20.9.0.

Para actualizar GitHub Pages:
1. Subir todos los archivos de esta carpeta.
2. Reemplazar los archivos existentes.
3. Confirmar el commit.
4. Esperar 1 a 3 minutos.
5. En iPhone, cerrar y volver a abrir la app. Si conserva el diseno anterior, eliminarla de inicio e instalarla nuevamente desde Safari.

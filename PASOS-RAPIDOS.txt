LUBAYD SA V22.1.0 - PASOS RÁPIDOS

1) Subir a GitHub todos los archivos y carpetas del proyecto.

2) En Cloud Shell:

cd ~/LUBAYD-SA-PRUEBA
git pull origin main
npm --prefix functions install
node --check functions/index.js
firebase deploy --only functions,firestore:rules --project lubayd-sa
firebase functions:list --project lubayd-sa

3) En el celular abrir:
https://TU-USUARIO.github.io/TU-REPOSITORIO/reset.html

4) Pulsar “Limpiar caché y abrir la aplicación”.

5) Iniciar sesión con internet una vez.

6) Entrar en Modo sin conexión y pulsar “Probar ubicación GPS”.

7) Probar llegada: obtener GPS, tomar foto y guardar.

8) Probar modo avión y luego sincronizar.

NO BORRAR LOS DATOS DEL SITIO si existen registros pendientes.

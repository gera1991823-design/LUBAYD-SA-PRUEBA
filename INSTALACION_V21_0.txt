LUBAYD SA V21.0 - TANQUE, TRAILER Y MAQUINAS
================================================

1. SUBIR TODOS LOS ARCHIVOS INTERIORES DEL ZIP A LA RAIZ DE GITHUB.
2. REEMPLAZAR LOS ARCHIVOS ANTERIORES.
3. EN CLOUD SHELL:

   cd ~/LUBAYD-SA-PRUEBA
   git pull origin main
   npm --prefix functions install
   firebase deploy --only "functions:syncFuelFlowRecord,functions:getFuelFlowState" --project lubayd-sa
   firebase deploy --only firestore:rules --project lubayd-sa

4. VERIFICAR:

   firebase functions:list --project lubayd-sa

5. EN ADMINISTRACION > MAQUINAS, CREAR O REVISAR LAS 3 MAQUINAS.
6. ACTUALIZAR LA PWA SIN BORRAR DATOS PENDIENTES.
7. CONFIRMAR VERSION 21.0.0.
8. REGISTRAR COMO PRIMER MOVIMIENTO EL SALDO REAL DEL TANQUE.

FLUJO
-----
PROVEEDOR -> TANQUE GENERAL -> TRAILER -> MAQUINAS

TODOS LOS MOVIMIENTOS REQUIEREN FOTO.
LOS OPERADORES REGISTRAN.
ADMINISTRADORES Y SUPERVISORES VISUALIZAN.

IMPORTANTE
----------
LA V21.0 UTILIZA COLECCIONES NUEVAS. LOS REGISTROS DE COMBUSTIBLE DE VERSIONES
ANTERIORES NO SE ELIMINAN, PERO NO SE MEZCLAN CON EL NUEVO FLUJO.

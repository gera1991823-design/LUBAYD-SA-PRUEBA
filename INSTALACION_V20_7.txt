LUBAYD SA V20.7 - ACCESO OFFLINE Y SESION DE 1 HORA
====================================================

1. SUBIR TODOS LOS ARCHIVOS A LA RAIZ DE GITHUB PAGES.
2. PUBLICAR firestore.rules EN FIREBASE.
3. DESPLEGAR LAS FUNCTIONS:

   firebase login
   firebase use lubayd-sa
   cd functions
   npm install
   cd ..
   firebase deploy --only functions

4. ELIMINAR E INSTALAR NUEVAMENTE LA PWA.
5. INICIAR SESION ONLINE COMO ADMINISTRADOR.
6. ABRIR USUARIOS.
7. ASIGNAR UN PIN A CADA OPERADOR Y PULSAR PREPARAR.
8. NOMBRAR EL DISPOSITIVO Y PULSAR HABILITAR DISPOSITIVO.
9. ACTIVAR MODO AVION.
10. PROBAR EL ACCESO LOCAL Y UNA MARCA.

LA SESION DURA 60 MINUTOS Y SE CIERRA AUTOMATICAMENTE.
EL ADMINISTRADOR NO MARCA: SOLO VISUALIZA, EDITA O ELIMINA.

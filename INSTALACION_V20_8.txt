LUBAYD SA V20.8.7 - INSTALACIÓN
================================

1. Subir todos los archivos interiores del ZIP a la raíz del repositorio GitHub.
2. Esperar la publicación de GitHub Pages.
3. En Cloud Shell ejecutar:

cd ~/LUBAYD-SA-PRUEBA
git pull origin main
npm --prefix functions install
firebase deploy --only "functions:authorizeOfflineDevice,functions:syncOfflineAttendance,functions:syncOfflinePart" --project lubayd-sa
firebase deploy --only firestore:rules --project lubayd-sa
firebase functions:list --project lubayd-sa

4. En cada celular, abrir la versión nueva con internet e iniciar sesión una vez con el operador.
5. Desde ese momento el operador puede usar ese celular con o sin conexión.
6. No borrar la PWA ni sus datos si existen partes pendientes.

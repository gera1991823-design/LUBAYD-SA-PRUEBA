LUBAYD SA V20.8.9 - INSTALACIÓN
================================

1. DESCOMPRIMIR EL ZIP.
2. SUBIR TODOS LOS ARCHIVOS INTERIORES A LA RAÍZ DEL REPOSITORIO.
3. REEMPLAZAR LOS ARCHIVOS ANTERIORES.
4. EN CLOUD SHELL EJECUTAR:

   cd ~/LUBAYD-SA-PRUEBA
   git pull origin main
   npm --prefix functions install
   firebase deploy --only "functions:syncFuelRecord" --project lubayd-sa

5. NO ES NECESARIO MODIFICAR LAS REGLAS SI YA ESTABAN PUBLICADAS LAS DE V20.8.8.
6. ACTUALIZAR LA PWA SIN BORRAR DATOS SI EXISTEN MOVIMIENTOS PENDIENTES.
7. CONFIRMAR QUE LA APLICACIÓN INDIQUE V20.8.9.

PRUEBA:
- INICIAL 150 L.
- CONSUMO 130 L => SALDO 20 L.
- NUEVA CARGA 180 L => SALDO 200 L.

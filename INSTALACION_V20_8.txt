LUBAYD SA V20.8.8 - INSTALACIÓN

1. Descomprimir el ZIP completo.
2. Subir todos los archivos interiores a la raíz del repositorio GitHub.
3. En Cloud Shell ejecutar:

   cd ~/LUBAYD-SA-PRUEBA
   git pull origin main
   npm --prefix functions install
   firebase deploy --only "functions:syncFuelRecord" --project lubayd-sa
   firebase deploy --only firestore:rules --project lubayd-sa

4. Comprobar:

   firebase functions:list --project lubayd-sa

5. En los celulares, abrir la aplicación con internet y pulsar Configuración > Actualizar aplicación.
6. No borrar datos de la PWA si hay registros pendientes.

# Lubayd SA V21.1.1 - Descanso móvil simple

Archivos modificados:

- `index.html`
- `attendance.css`
- `service-worker.js`

No requiere desplegar Functions ni reglas de Firestore.

## Cambios

- Vista móvil compacta para operadores.
- Estado y duración en una sola tarjeta.
- Botones grandes para iniciar y finalizar descanso.
- Los datos de foto, GPS y horarios permanecen visibles en tarjetas simples.
- Se oculta el historial personal en móvil para reducir desplazamiento.
- Cámara adaptada a pantalla completa en teléfonos.

## Publicación

Reemplazar los tres archivos en GitHub, esperar GitHub Pages y actualizar la PWA desde Configuración.
